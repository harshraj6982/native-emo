
# Food Delivery App 

It is a Food Delivery App like Uber Eats.
## Installation

Clone the project

```bash
  git clone https://github.com/anuDev29/native-emo
```

Go to the project directory

```bash
  cd native-emo
```

Install dependencies

```bash
  npm install
```


## Run Locally

Start the Expo

```bash
  npm run start
```

Start Expo with Android

```bash
  npm run android
```

Start Expo with IOS

```bash
  npm run ios
```

Start Expo with Web

```bash
  npm run web
```
